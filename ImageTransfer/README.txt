C#    .Net Framework 4.6.1   VS2017

程序所在路径请尽量不要出现中文

双击ImageTransfer.exe即可运行，或在.\ImageTransfer\ImageTransfer\bin\Release目录下也可找到exe

进入程序首先导入图片，选择插值方式（请只选择一种）,选择要执行的操作（旋转或畸变），在滑块拖动控件中
给出选择的参数值：
旋转半径：图片最大半径（边长一半）的百分比
旋转角度：最大角度可选360度
畸变半径：图片最大半径（对角线一半）的倍数

点击输出图片可观察结果，图片可保存

srcpic是测试图片

outpic是输出样例图片
